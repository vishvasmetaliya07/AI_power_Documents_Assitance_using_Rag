# 📄 AI-Powered Documents Assistant using RAG

An AI-powered application that lets users upload documents and ask questions in natural language. The system uses **Retrieval-Augmented Generation (RAG)** to retrieve relevant information from the document and generate accurate, context-aware answers.

## ✨ Features

* 📄 Upload PDF documents
* 💬 Chat with your documents
* 🔍 Semantic search using vector embeddings
* 🤖 AI-powered question answering
* ⚡ Fast and accurate responses
* 🎨 Simple Streamlit interface

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Google Gemini
* FAISS / ChromaDB
* HuggingFace Embeddings
* PyPDF2
